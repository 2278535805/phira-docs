# 图片元素 `img`

```js
img(id, url, r, c, t)
```

- `id`(选填): 元素 ID
- `url`(必填): 图片 URL, 用双引号括起来
- `r`(必填): 图片显示的位置矩形, 类型为 `Rect`
- `c`(选填, 默认为 `"white"`): 在图片上叠加的颜色. 若颜色半透明, 图片也会半透明
- `t`(选填, 默认为 `cropCenter`): 图片的裁剪方式. 可选值有:
  - `cropCenter`: 保持图片比例, 从图片中心裁剪，保证 `r` 被完全填满
  - `inside`: 保持图片比例, 使图片完全显示在 `r` 中
  - `fit`: 拉伸为 `r` 的大小
